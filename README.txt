Note: 
To run this app first download Trained data for Tesseract library for the following link.

"https://code.google.com/p/tesseract-ocr/downloads/detail?name=tesseract-ocr-3.02.eng.tar.gz&can=2&q="

Extract and save the folder in the sd card of the device under'Pictures'as 'Pictures/tesseract-ocr'

For more information click on "Trained Data" button in the App.
